
# Proyecto React.js (e-commerce)


Este es el trabajo final de la carrera de Desarrollo Frontend React de Coderhouse.

Se utilizó React como herramienta principal para crear un e-commerce de venta de instrumentos y herramientas musicales, incorporando también  conocimientos obtenidos de los cursos de Desarrollo Web y Javascript.




## Herramientas

 - React.js: Como biblioteca principal para crear la interfaz de usuario.
 - React Router Dom: Para la navegación dentro de la web.
 - React Hook Form: Para generar los formularios de compra.
- Context: Para optimizar el funcionamiento de código.
- HTML: Para la estructura de la página
- CSS: Para los estilos de la web
- JS y JSX: Como lenguajes para desarrollar la lógica de la aplicación.


## Instalacion

Colnar el repositorio con:```
bash git clone https://github.com/francorp1993/EntregaProyectoFinal-PiacentiniFrancoReact.git```



Instalar las dependencias:  ```bash
 npm install```

Inicializar la app:```
 npm run dev```
